中央通訊社
2016/10/09 18:55:03
（中央社記者田裕斌台北2016年10月9日電）由中信金控(2891)發起的「Home Run Taiwan愛就跑！9天8夜不斷電環台馬拉松」今天進入尾聲，第100棒跑者找來國內超馬好手林義傑鼎力相助。中信金董事長顏文隆表示，中信金在50年前由19名員工、120坪的辦公室在台北市館前路發跡，能有今天的規模，主要歸功於國家總體經濟的成長、社會大眾、客戶的信賴與支持，及同仁的無私付出，選擇以不一樣的方式慶生，邀請社會各界以環台路跑的方式看故鄉，凝聚對台灣的愛。具有象徵意義的第100棒今年下午3時從台北市迎風河濱公園出發，由超馬好手林義傑、長期倡導路跑活動的抗癌勇者吳傳興及10位民眾、9位今年剛好50歲的中信金員工組成。中信金表示，這次台灣史上第1次200小時不斷電接力、共計1000公里的「Home Run Taiwan愛就跑！9天8夜不斷電環台馬拉松」號召1600名跑者參加，從台灣頭到台灣尾。同時，今年元月上線的「Home Run Taiwan」粉絲團以紀錄片的方式刻畫出台灣各角落默默努力的小人物故事，有30個真實故事將在粉絲團中放送。

中央通訊社
2016/10/03 20:22:05
（中央社記者田裕斌台北2016年10月3日電）「Home Run Taiwan愛就跑！9天8夜不斷電環台馬拉松」活動邁入第3天，中信慈善基金會董事長辜仲諒熱血加入第28棒，與花蓮富源國小棒球隊12位小球員一起跑。發起單位中信金控（2891）表示，「Home Run Taiwan愛就跑！9天8夜不斷電環台馬拉松」，第3天來到山明水秀的花蓮縣，中信慈善基金會董事長辜仲諒意外現身，下午參與路跑，與富源國小棒球隊自馬太鞍橋旁的獅子公園出發，一同跑第28棒，互動熱絡，現場互相加油打氣。辜仲諒表示，不同的愛有不同的跑法，來到花東就是為了支持富源國小這一群愛棒球的孩子而跑，這群孩子是中信慈善基金會扶助的，他與孩子們曾約定要一起跑步，今天藉由環台馬拉松實現他對孩子們的諾言，他也呼籲社會一同感謝臺灣這塊土地以及默默付出的無名英雄，希望大家懷抱信念，了解自己為什麼而跑。富源國小棒球隊每周都繞著操場跑10圈，對這12位小球員而言，陪跑3.6公里的路程並不困難，但路程中天氣驟變，突然下起了一陣大雨，一群人即使淋濕了，仍冒雨前行，其中遇到上坡路段，還有小朋友相互合作、互揹一程。中國信託商銀表示，花蓮富源國小是中信慈善基金會「愛接棒．中信少棒資助計畫」的資助學校之一，日前「Home Run Taiwan」粉絲團邀請網友分享影片即贈球偏鄉少棒的活動，富源國小亦列於受贈名單；富源國小棒球隊教練張茂三指出，練習球屬消耗性器材，這些資源對小球員練球幫助很大，感謝中國信託長期對偏鄉少棒的支持和培育，對於此次路跑，學校樂於共襄盛舉。此外，花蓮路跑路段，上午還有阿美族勇士加入跑第25棒，其中一位跑者張吉祥是中信銀員工，也是壽豐鄉土生土長的阿美族，2008年曾在澳門參加國際半程馬拉松，獲得第2名的成績，8年後專程返鄉，為公益和家鄉而跑，象徵不分族群、人人皆為台灣這塊孕育我們的土地而努力。

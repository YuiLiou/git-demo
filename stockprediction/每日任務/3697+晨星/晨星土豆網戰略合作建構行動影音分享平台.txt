蘋果日報
2012-06-21 
【陳建彰╱台北報導】F-晨星（3697）昨天宣布，與中國領導視頻分享業者土豆網進行戰略合作，未來將整合土豆網豐富的影音娛樂內容及分享管道，與晨星領先支持的電影王應用，打造新型行動影音娛樂分享平台，進一步整合在晨星的智慧型手機解決方案。晨星持續強化行動通訊平台的硬體規格及軟體應用，已於日前發表可同時支援TD-SCDMA（Time Division-Synchronous Code Division Multiple Access，分時-同步分碼多工存取）╱WCDMA（Wide band Code Division Multiple Access，寬頻分碼多工存取）規格的智慧型手機方案，提升行動通訊產品組合廣度，昨天與土豆網合作，強化軟體應用內容，加強智慧型手機解決方案的附加價值。晨星全球手機事業部總經理鄧矣兵表示，晨星擁有堅強的研發團隊，除不斷提升行動通訊主晶片集成度、協助客戶降低系統成本以強化產品競爭力，更持續領先市場、運用其在影音多媒體方面的經驗，透過穩定軟體開發能力，開拓創新產品差異化特色。這次透過與中國視頻分享領導業者土豆網合作，客戶不僅能藉由晨星支援的電影王應用平台，推出更具競爭力且定位精準的平台方案，更能提供終端消費市場優質行動影音體驗。土豆網無線拓展副總裁吳錚表示，為讓用戶體驗更快捷及豐富的移動視頻內容，土豆網通過多種方式為手機用戶提供服務，此次與晨星合作，將土豆用戶端直接集成在手機晶片中，是移動視頻發展的重要里程碑。
